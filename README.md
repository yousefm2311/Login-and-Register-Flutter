# Login And Register Flutter

A Flutter authentication UI project focused on login, registration, and form-based onboarding screens.

## Status

Public learning project.

## Features

- Login screen UI
- Registration screen UI
- Form-based authentication flow foundation
- Mobile-first Flutter layout
- Existing screenshots included below

## Tech Stack

- Flutter
- Dart

## Getting Started

```bash
git clone https://github.com/yousefm2311/Login-and-Register-Flutter.git
cd Login-and-Register-Flutter
flutter pub get
flutter run
```

## Environment Variables

Create local configuration only if authentication is connected to external services.

```env
API_BASE_URL=
FIREBASE_PROJECT_ID=
```

Never commit real credentials or production secrets.
